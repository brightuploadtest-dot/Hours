How do I run this app when my computer is off?

Why it turns off: Mobile browsers block offline capabilities (Service Workers) on plain HTTP local network connections (e.g. http://192.168.x.x). Caching is strictly disabled.

The 2-Minute Premium Setup:

  1. Host on GitHub Pages (Free & Secure): Create a free GitHub repository, upload these files (excluding database.json and server.js), and turn on Pages in settings. You will get a secure HTTPS link like https://username.github.io/hour-tracker.
  2. Open on Phone & Add to Home Screen: Open that HTTPS link. Your phone will successfully register the Service Worker. The app will now **load in 0.1s and work 100% offline, even on a plane or when your PC is shut off!**
  3. Link to your PC database: Enter your computer's local IP (e.g., http://192.168.4.23:8000) in the input above. Whenever your PC is on, your phone will sync silently and merge all offline edits!
Tip: Run Server on Windows Startup
To make your PC sync server run automatically on boot, press Win + R, type shell:startup, and copy a shortcut of start_sync_server.bat into that folder!

Export your database as a file to transfer it manually (via Email, WhatsApp, etc.), or import a backup file to restore.
